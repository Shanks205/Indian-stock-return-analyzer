## Key modelling lesson by company

| Company | Main modelling lesson |
|---|---|
| Hero MotoCorp | Do not overpay for a mature quality business by assuming every optionality works. |
| Natco Pharma | Normalize product-cycle earnings and separate pipeline optionality from base cash flows. |
| NALCO | Use mid-cycle commodity margins instead of extrapolating peak aluminium/alumina conditions. |
| Hindustan Zinc | Separate strong zinc/silver cycles from sustainable mid-cycle mining profitability. |
| CPCL | Model refining businesses around normalized GRM, not reported revenue growth alone. |
| Sharda Cropchem | Test accounting-profit recovery against working-capital and receivable cash conversion. |
| Fiem Industries | Test margin improvement against customer concentration and two-wheeler-cycle risk. |
| Time Technoplast | Test value-added growth against capex, raw-material and working-capital requirements. |
| Force Motors | Separate recurring operating profit from exceptional gains and auto-cycle peaks. |
| Triveni Turbine | Test order-book quality against execution, margin sustainability and working-capital conversion. |

## Common DCF discipline across all companies

The project applies the following common valuation discipline:

- Use FCFF / DCF as the primary intrinsic value framework.
- Interpret output as a valuation range, not a precise target price.
- Use scenario analysis for bear, base and bull cases.
- Use WACC vs terminal-growth sensitivity.
- Add business-specific sensitivity tables.
- Separate operating thesis from valuation discipline.
- Avoid extrapolating one strong quarter permanently.
- Separate recurring operating performance from one-off or cycle-driven gains.
- Record all assumptions in an assumptions log.
- Record all source and refresh dependencies in a source log.
