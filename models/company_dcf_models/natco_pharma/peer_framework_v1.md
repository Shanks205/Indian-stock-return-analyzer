# Natco Pharma Peer Framework v1

## Purpose

This file completes the peer-comparison layer for the Natco Pharma public-source v1 research pack.

Natco should not be compared mechanically with large diversified pharma companies because its earnings are more concentrated and product-cycle driven. Peer comparison should be used as a sanity check alongside the DCF, not as a replacement for intrinsic valuation.

## Interpretation framework

### Natco vs Divi's Laboratories

Divi's is useful as a high-quality export/manufacturing benchmark, but Natco's niche product-cycle exposure makes its earnings less predictable. Divi's may deserve a different valuation multiple because of scale and perceived quality.

### Natco vs Laurus Labs

Laurus helps frame cyclicality and margin compression risk. It is useful when evaluating whether Natco's high-margin periods should be normalized.

### Natco vs Granules India

Granules provides a generics/API reference point. However, Natco's specialty and complex-products strategy can justify different margins and risk assumptions.

### Natco vs Ajanta Pharma

Ajanta provides a branded/specialty comparison and may be useful for domestic formulation stability.

### Natco vs IPCA Laboratories

IPCA provides a broader Indian pharma comparison with regulated-market and domestic exposure.

### Natco vs Sun Pharma

Sun Pharma should be used only as a large-cap pharma reference. It should not mechanically anchor Natco's fair value because scale, diversification, and risk profile differ significantly.
