## Possible interview questions

### 1. Why did you calculate daily returns?

Daily returns help measure how much the stock price changes each trading day. They are the base input for calculating volatility, cumulative return, and risk-adjusted performance.

### 2. What does volatility mean?

Volatility measures how much returns fluctuate. A stock with high volatility moves more sharply up and down, which usually means higher risk.

### 3. What does Sharpe ratio tell us?

Sharpe ratio tells us how much return is earned for each unit of risk. A higher Sharpe ratio generally means better risk-adjusted performance.

### 4. What is maximum drawdown?

Maximum drawdown shows the worst fall from a previous peak. It helps investors understand downside risk.
