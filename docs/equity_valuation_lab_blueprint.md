# Indian Equity Valuation Lab — Project Blueprint

## Purpose

This repository will evolve from a basic stock-return analyzer into a practical investment-banking-style equity valuation project for Indian listed companies.

The project is designed for three goals:

1. Build a strong GitHub portfolio project for finance, equity research, investment banking, and portfolio analytics roles.
2. Create a repeatable DCF and fundamental screening framework for Indian equities.
3. Build the foundation for a future SSRN-style working paper on Indian equity valuation using DCF, Piotroski F-Score, margin of safety, and barbell portfolio construction.

## Core research question

Can a disciplined framework combining accounting quality, intrinsic valuation, margin of safety, and portfolio-fit rules improve Indian equity selection compared with simple price-return analysis?

## Working title

**From Screening to Intrinsic Value: A Practical Fundamental Research Framework for Indian Stocks**

## Investment banking analyst workflow

The project follows a practical analyst workflow:

1. Define the investment universe.
2. Screen companies using quality, value, growth, and financial health filters.
3. Calculate accounting quality indicators such as Piotroski F-Score.
4. Select companies for DCF valuation.
5. Build historical financial statement analysis.
6. Forecast operating drivers.
7. Estimate WACC and terminal value.
8. Perform base, bear, and bull scenario valuation.
9. Run sensitivity analysis for WACC, terminal growth, revenue growth, margin, and reinvestment.
10. Compare intrinsic value with market price.
11. Apply margin-of-safety rules.
12. Rank companies using valuation, quality, growth, risk, and portfolio-fit scores.
13. Produce a final research dashboard and written analyst summary.

## Research discipline

Every valuation should include:

- source of historical numbers
- source of market price
- source of risk-free rate
- WACC calculation assumptions
- growth-rate methodology
- margin assumptions
- terminal-growth logic
- risk-regime check
- AI attention-bias check
- portfolio-fit conclusion
