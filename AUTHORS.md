# Authors

- **Himanshu Dabi**

## Project focus

Indian Equity Valuation Lab combines:

- Indian equity screening
- Piotroski F-Score
- DCF valuation
- WACC stress testing
- margin-of-safety analysis
- barbell portfolio construction
- analyst-style company research
- future SSRN-style research paper development

## Contribution principle

Both authors should make independent GitHub commits where possible so that GitHub records each person's contribution history.

Recommended workflow:

1. Create issues for tasks.
2. Assign work between Himanshu and Hitesh.
3. Each author works on a separate branch.
4. Each author opens pull requests from their own GitHub account.
5. Pull requests are reviewed and merged into the main branch.

This keeps the project transparent, professional, and credible for resumes, interviews, and future research-paper authorship.
