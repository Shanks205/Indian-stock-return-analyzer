# AGENTS.md

## Project purpose

This repository is a finance project for learning Python, GitHub, and equity analytics.

## User background

The user is a finance aspirant learning Python, SQL, GitHub, and quantitative finance.

## Style rules

- Keep the code beginner-friendly.
- Explain financial formulas clearly.
- Use simple comments where calculations may confuse beginners.
- Avoid unnecessary complexity.

## Finance rules

- Do not give investment recommendations.
- Use sample or public market data only.
- Clearly explain assumptions.
- Validate calculations where possible.

## GitHub rules

- Keep README professional and recruiter-friendly.
- Include installation instructions.
- Include methodology.
- Include limitations.
- Include future improvements.
